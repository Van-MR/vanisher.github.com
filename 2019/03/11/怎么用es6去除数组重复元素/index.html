怎么用es6去除数组重复元素

Posted on 2019-03-11

这里有三种方法去过滤数组中的重复元素,经过滤后数组的元素都具有唯一性,我最喜欢的方式是使用Set,它最简短和简单

1.使用Set
首先来解释一下什么是Set

Set是ES6新的数据对象,在Set集合中只允许你储存独一的值不允许存在两个相同的值,所以当我们传入一个数组的时候Set会自动去除里面重复的元素

好了,让我们回到代码中,分解下到底发生了什么,首先我们创建一个Set并且传入array，因为Set中不允许存在重复的值,数组中重复的元素会被remove掉,然后我们将经过Set处理后的值转换成新的数组通过对象展开运算符…

我们同样可以将Set转换成Array用 Array.from()方法

2.使用filter过滤
为了去理解这里的选项,让我们看下这两个方法到底在做什么: indexOf 和 filter

indexOf()

indexOf方法通常用来返回元素第一次出现的索引位置

filter

filter方法根据我们的条件去创建一个新的数组,换句话说传入的元素如果满足条件就会被包含在过滤后的数组中,反之不满足条件的将不会被included

3.使用reduce()
reduce方法通常被用于将数组内的多个元素进行combine操作,最终的结果是根据我们传入的reducer函数决定的
